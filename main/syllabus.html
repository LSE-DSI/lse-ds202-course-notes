📓 Syllabus

A list of what will happen every week.

Click on each Week’s link for more information (slides, lab instructions, recommended resources, etc.).

Intro
🗓️ Week 01 Lecture Introduction, Context & Key Concepts
Lab No class this week.
(Use this time to revisit basic R programming)
Supervised Learning
🗓️ Week 02 Lecture Simple and Multiple Linear Regression
Lab Revision of R: data structures, basic commands and some tidyverse
🗓️ Week 03 Lecture Classifiers (Logistic Regression & Naive Bayes)
Lab Linear Regression (James et al. 2021, chap. 3)
🗓️ Week 04 Lecture Resampling methods
Lab Classification Methods (James et al. 2021, chap. 4)
🗓️ Week 05 Lecture Non-linear algorithms (SVM & tree-based models)
Lab Cross-Validation and the Bootstrap (James et al. 2021, chap. 5)
Summative
  • Worth: 20% of final marks
  • Problem set involving all content covered until Week 05
  • Deadline: day before next lab
Unsupervised Learning
🗓️ Week 07 Lecture Unsupervised Learning: Clustering
Lab Unsupervised Learning: Clustering (James et al. 2021, chap. 12)
🗓️ Week 08 Lecture Unsupervised Learning: Dimensionality Reduction
Lab Unsupervised Learning: PCA (James et al. 2021, chap. 12)
Summative
  • Worth: 20% of final marks
  • Problem set about non-linear regression and clustering
  • Deadline: day before next lab
Applications
🗓️ Week 09 Lecture Applications: Predictive Modelling on Tabular Data
Lab We will create machine learning models using a selected real tabular dataset
🗓️ Week 10 Lecture Applications: Text as Data & Topic Modelling (with guest speakers)
Lab We will explore unsupervised models using a couple of text datasets
🗓️ Week 11 Lecture Applications: Social Media Data (with guest speakers)
Lab We will create ML models using the data provided by (Sumner et al. 2012)
followed by a discussion on explainability and potential ethical issues
Summative
  • Worth: 20% of final marks
  • Problem set about unsupervised learning applications
  • Deadline: end of Week 11+1

References

James, Gareth, Daniela Witten, Trevor Hastie, and Robert Tibshirani. 2021. An Introduction to Statistical Learning: With Applications in R. Second edition. Springer Texts in Statistics. New York NY: Springer. https://www.statlearning.com/.
Sumner, Chris, Alison Byers, Rachel Boochever, and Gregory J. Park. 2012. “Predicting Dark Triad Personality Traits from Twitter Usage and a Linguistic Analysis of Tweets.” In 2012 11th International Conference on Machine Learning and Applications, 2:386–93. https://doi.org/10.1109/ICMLA.2012.218.
